## Requirements
- The modules package requires:
    - PHP 8.0 or higher
    - MySql 5.7 or higher

## Install
### Create project with composer
```
composer create-project --prefer-dist juzaweb/juzacms blog
```
### Install

Config database in your `.env` file, and run:

```
php artisan juzacms:install
```
